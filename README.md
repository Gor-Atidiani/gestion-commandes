# GestionCommandes

Projet de démonstration orienté backend C#/.NET, développé dans le cadre d’un plan de progression sur 60 jours.

## Objectif

Développer une API complète ASP.NET Core en partant des fondamentaux (POO, async, structure en couches, base de données, sécurité).

## Semaine 1 - POO et base console

- Création des entités `Client` et `Commande`
- Implémentation d'une interface `IExportable`
- Architecture de projet orientée "clean"

## Stack technique

- C# 9 / .NET 6
- Visual Studio
- Git + GitHub
